# 基本语句
## 输出语句
### print
``` ruby
print "hello..."
print "hello..."
---
hello...hello...
```
### puts
``` ruby
puts "hello..."
puts "hello..."
---
hello...
hello...
```
### p
``` ruby
p "hello..."
p "hello..."
---
"hello..."
"hello..."
```

## 字符串中变量引用
``` ruby
user_name = "yxy"
puts "邢台第一刀客：#{user_name}"
---
邢台第一刀客：yxy
```
